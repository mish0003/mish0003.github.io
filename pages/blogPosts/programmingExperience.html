My experience

  • I started my programming journey in 2022, learning through the Bulgarian National IT Career Program, where I built console applications and worked on ASP.NET projects, including Razor Pages. Around the same time, I also began learning HTML and CSS, which allowed me to create and style static web pages.

    Later in 2022, I took several online courses at SoftUni, where I focused on JavaScript and expanded into back-end development with Node.js and Express.js. This gave me the foundation for building interactive web applications and APIs.

    In 2023, I continued with SoftUni courses and advanced into TypeScript and Angular, which improved my ability to build scalable and type-safe applications. Alongside this, I became familiar with tools like Git & GitHub for version control and Postman for API testing.

    In 2024, I explored more modern technologies such as React for building user interfaces and MySQL Workbench for database design and management. By then, I already had experience with SQL Server (SSMS) and MongoDB, which gave me both relational and NoSQL database knowledge.

    For development environments, I mainly use Visual Studio for C# and ASP.NET projects and Visual Studio Code for front-end and JavaScript development.
